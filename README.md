## > Descripción

Se ha desarrollado una versión del clásico juego *Hundir la Flota* en Python, aplicando Programación Orientada a Objetos (POO) y separación de responsabilidades.

El objetivo del juego es destruir todos los barcos del rival antes que él.

---

## > Estructura del proyecto

- `variables.py` → constantes del juego  
- `clases.py` → clase `Tablero`  
- `funciones.py` → lógica del juego  
- `visualizacion.py` → representación del tablero en consola  
- `main.py` → integración y flujo principal del juego
